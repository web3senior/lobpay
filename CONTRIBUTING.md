# Contributing to LobPay SDK

Thank you for your interest in contributing to the X402 SDK for the LUKSO blockchain. We welcome contributions from the community to help make this SDK more robust and efficient.

## Code of Conduct
By participating in this project, you agree to abide by our Code of Conduct. Please be respectful and professional in all interactions.

## How to Contribute

### Reporting Bugs
- Check the issue tracker to see if the bug has already been reported.
- If not, open a new issue with a clear title and description.
- Include steps to reproduce the bug and any relevant logs or screenshots.

### Suggesting Enhancements
- Open a new issue with the tag "enhancement".
- Describe the feature you'd like to see and why it would be useful for the LUKSO ecosystem.

### Pull Requests
- Fork the repository and create a new branch from `main`.
- Ensure your code follows the established style guidelines.
- Add comments to your code as per project conventions (note: do not start comments with numbers).
- Include tests for any new functionality or bug fixes.
- Submit a pull request with a detailed description of your changes.

## Development Setup
- Clone your fork: `git clone https://github.com/web3senior/lobpay.git`
- Install dependencies: `npm install` or `yarn install`
- Run tests: `npm test`

## Style Guidelines
- Use descriptive variable and function names.
- Keep functions small and focused on a single task.
- Ensure all public APIs are well-documented.

## Community
Join our community channels to discuss the project and get help from other contributors.
